# Code of Conduct

This project aims to be a practical, respectful place for people building local agent tooling.

## Expected Behavior

- Be direct, kind, and specific.
- Critique ideas, code, and documentation rather than people.
- Assume good intent, but accept correction when something causes harm.
- Keep security reports discreet and constructive.

## Unacceptable Behavior

- Harassment, threats, personal attacks, or discriminatory language.
- Publishing another person's private data, credentials, logs, or local paths.
- Using issues or pull requests to pressure maintainers into unsafe disclosure.

## Enforcement

Maintainers may edit, hide, or remove comments; close issues; block participants; or take other reasonable action to keep the project healthy.
